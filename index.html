APEX Trading Agent — Autonomous Crypto AI
Autonomous crypto trading agent powered by Claude AI that analyzes live Hyperliquid perpetuals market data and makes real-time BUY/SELL decisions with full rationale.

How it works: Fetches live prices, 24h volume, funding rates, and open interest for 12 assets every 15 seconds from Hyperliquid's API. IoT · Backend · Full-Stack
LotOwl — Smart Parking Availability System
Built with a 3-person team at HackYSU 2026. LotOwl reduces the time drivers spend searching for campus parking by combining IoT sensors, a Flask REST API, and an interactive map interface into one real-time system.

The system has three layers: HC-SR04 ultrasonic sensors on WiFi-enabled ESP32 microcontrollers detect vehicle events and POST accuracy-scored data to the backend. The Flask API processes these events, determines entry/exit logic, updates occupancy counts, and serves lot availability to the frontend. A Leaflet.js web interface displays color-coded availability markers on an interactive campus map with navigation support.

My role — Backend: Designed the full Flask API from scratch. Built the POST /update endpoint for sensor ingestion, GET /lots and GET /summary for frontend consumption, thread-safe JSON storage with atomic file writes, and parking lot data model with lot_id, capacity, occupancy, and timestamps.
